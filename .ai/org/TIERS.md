## Activation Rule

**Do not activate a Tier 2+ agent until there is a specific need.**

The default operating team for any feature sprint is:
```
Hermes → OpenClaw → ChatGPT (QA) → Sammi (Approve)
```

Only bring in Claude when design work is needed.
Only bring in Gemini when research is needed.
Only bring in Codex when infrastructure changes are needed.
