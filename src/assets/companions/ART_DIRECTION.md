# Companion Art Direction

Use this direction for future Barbie / Gym Bro / companion body evolution assets.

## Core Style

- Cute soft 3D cartoon character
- Big expressive eyes
- Rounded youthful face proportions
- Warm animated-movie feeling
- Cozy premium wellness app aesthetic
- Soft pastel palette
- Gentle, emotionally warm, non-aggressive

## Reference Feeling

The user likes the soft animated heroine feeling of:

- expressive round eyes
- warm cinematic light
- soft face shape
- cute but premium cartoon rendering
- friendly emotional expression

Do not copy any specific existing character, outfit, face, scene, or IP.

## Body Evolution Rule

Body changes must be visible, but still cute:

- Stage 1: cozy beginner, soft rounded silhouette, oversized outfit
- Stage 2: clearer mid-progress, more upright, slightly more toned / stronger
- Stage 3: clear glow-up, visibly healthier / stronger, but still soft cartoon and wholesome

Avoid making stage 3 look like a realistic fitness influencer, harsh gym model, or aggressive bodybuilder.

## Prompt Base

Use this prompt base when regenerating:

```text
Create an original cute wellness companion character in a soft 3D animated movie style.
Use big expressive eyes, rounded face, warm cinematic lighting, pastel wellness colors, and cozy premium self-care app energy.
Show 3 body evolution stages side-by-side with the same character identity.
Stage 1 should feel cozy beginner with a soft rounded silhouette and oversized outfit.
Stage 2 should show clear mid-progress with more upright posture and slightly more toned/stronger proportions.
Stage 3 should show a clear glow-up with visibly healthier/stronger body lines, while staying cute, wholesome, modest, and non-aggressive.
Use a flat warm cream background, no text, no logo, no watermark.
Avoid realistic fitness influencer style, sexualized posing, harsh gym branding, extreme body ideals, or body shame.
```
